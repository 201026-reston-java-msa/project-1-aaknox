How to Use ReimburseMe - User Guide

Submit A New Reimbursement Request: Select this option to send a new reimbursement request to HR. Receipts are required (10 max).
View Past Requests: Select this option to view your past submitted requests (of APPROVED/DENIED status).
View Pending Requests: Select this option to view your active requests (of PENDING status).
View Profile: Select this option to view and update your personal information.

Note: For the "Adminstration" option how-tos, please see the help box in the adminstration home page.
If you want to cancel or update any active request, you can contact our finance team at contact.reimburseme@gmail.com.
